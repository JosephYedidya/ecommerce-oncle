# 🛍️ ecommerce-oncle-frontend

Une application web de gestion des stocks e-commerce avec deux rôles utilisateurs: **Admin** et **Caissière**.

## ✨ Fonctionnalités

### 👨‍💼 Admin
- ➕ Ajouter des produits avec variantes/types
- 📊 Consulter les statistiques (CA, ventes, valeur stock, produit top)
- 📋 Visualiser tous les produits et stocks
- 📜 Voir l'historique complet des ventes
- 🌙 Mode clair/sombre

### 👩‍💼 Caissière
- 🛒 Enregistrer les ventes rapidement
- 📦 Consulter les produits disponibles
- 🌙 Mode clair/sombre

## 🛠️ Stack Technique

- **Frontend:** React + Vite
- **UI Library:** Ant Design
- **State Management:** React Hooks
- **HTTP Client:** Axios
- **Styling:** CSS-in-JS + Ant Design themes
- **Deployment:** Vercel
